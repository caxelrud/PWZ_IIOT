# PWZ-IIOT

The objective of this project is to demonstrate a solution for Industrial Internet of Things using the following technologies:
   * Azure IOT Hub
   * OPC-DA (legacy systems) gateway
   * Azure Functions
   * F#
   * POWER BI
   * Data Compression
   * Client Buffering
